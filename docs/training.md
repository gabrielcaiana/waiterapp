# Guia de Treinamento Full Stack

Este guia foi criado para auxiliar no treinamento de equipes full stack usando o projeto WaiterApp como base.

## 📚 Módulos do Treinamento

### 1. Introdução ao Monorepo
- Conceitos de Monorepo
- Configuração do NX
- Estrutura do projeto
- Gerenciamento de dependências com pnpm

### 2. Backend com Node.js
- Arquitetura da API
- Integração com MongoDB
- Autenticação e Autorização
- Upload de arquivos com AWS S3
- Testes unitários e de integração

### 3. Frontend com React
- Estrutura do projeto web
- Gerenciamento de estado
- Integração com a API
- Componentização
- Testes com Jest e React Testing Library

### 4. Mobile com React Native
- Configuração do ambiente mobile
- Navegação
- Integração com a API
- Gerenciamento de estado
- Testes em dispositivos móveis

### 5. DevOps e Infraestrutura
- Containerização com Docker
- Docker Compose
- CI/CD
- Deploy
- Monitoramento

## 🎯 Exercícios Práticos

### Módulo 1: Backend
1. Criar uma nova rota na API
2. Implementar autenticação
3. Adicionar upload de imagens
4. Escrever testes unitários

### Módulo 2: Frontend
1. Criar uma nova página
2. Implementar formulários
3. Adicionar validações
4. Implementar testes

### Módulo 3: Mobile
1. Criar uma nova tela
2. Implementar navegação
3. Adicionar funcionalidades offline
4. Testar em diferentes dispositivos
